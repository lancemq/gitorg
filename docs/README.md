# Git Org Academy Docs

这个目录用于沉淀网站的产品需求、信息架构、页面设计原则与后续演进约定。

## 文档列表

- `product-requirements.md`
  说明网站的目标、目标用户、核心功能范围、内容范围和非功能要求。
- `information-architecture.md`
  说明频道结构、路由层级、左侧导航原则、搜索范围和内容组织方式。
- `design-guidelines.md`
  说明页面布局、视觉风格、交互约定、双语策略和内容展示规则。
- `roadmap.md`
  说明后续优化方向，包括内容深化、搜索增强、SEO、内容运营与体验迭代。

## 当前站点摘要

- 技术栈：Next.js App Router、TypeScript、MDX、Vercel Analytics
- 内容形态：双语内容站，中文和英文分别独立路由
- 当前内容主题：101 个唯一主题，202 篇双语 MDX 文档
- 核心频道：
  - Learning Path
  - Git Commands
  - Best Practices
  - Workflows
  - Git Internals
  - Git History
  - FAQ

## 使用方式

- 新增页面或频道前，先更新 `product-requirements.md` 和 `information-architecture.md`
- 调整导航、搜索、标题层级、卡片结构前，先参考 `design-guidelines.md`
- 规划下一轮扩容、优化或重构时，先更新 `roadmap.md`
