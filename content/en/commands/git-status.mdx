# git status Tutorial

## The short version

`git status` is the safest and most useful command for checking what is going on in your repository right now.

## What it tells you

- current branch
- staged changes
- unstaged changes
- untracked files
- ahead / behind state relative to the upstream branch

## Common forms

```bash
git status
git status --short
```
