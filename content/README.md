# Content Library

本目录用于维护 Git Org Academy 的教程素材，统一采用 Markdown / MDX 存储。

## 目录约定

- `content/zh/` 中文内容
- `content/en/` 英文内容
- 二级目录按主题拆分：
  - `learning-path/`
  - `commands/`
  - `workflows/`
  - `recovery/`
  - `concepts/`

## 文档规范

每篇 MDX 文档建议包含：

1. `export const metadata = { ... }`
2. 标题与摘要
3. 核心概念
4. 操作步骤或示例
5. 常见误区 / 风险提示
6. 来源链接
